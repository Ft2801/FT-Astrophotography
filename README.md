# Fabio Tempera - Astrophotography Portfolio

Questo è il repository ufficiale del portfolio di Astrofotografia di Fabio Tempera.

## Tecnologie

*   **HTML5 & CSS3** per la struttura e lo stile.
*   **Vanilla JavaScript (ES6+)** per la logica e gli effetti.
*   **HTML5 Canvas** per il background stellato interattivo.

## Features

*   Design "Deep Space" immersivo.
*   Background a parallasse reattivo a mouse e scroll.
*   Cursore custom.
*   Ottimizzazione SEO.
*   Sezione dedicata al software "Tstar".
